# 8 - Why would you run the command 'git status'?
To review the files you have modified but not yet staged for commit, which files
are staged, and the untracked files.

# 9 - What does the command 'git add .' do?
The 'git add .' will stage all the changes in the current folder and subfolders
if any, making them ready for the next commit.
